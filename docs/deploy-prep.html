오브젝트 스토리지 (광고 영상 업로드)

S3 vs R2 선택 레포는 이미 @aws-sdk/client-s3 씀. 두 옵션 다 같은 SDK로 동작:
AWS S3 — 안정, 다만 egress(다운로드) 트래픽 비용 큼. 광고 영상 자주 다운로드되면 청구서 늘어남.
Cloudflare R2 (강력 권장) — egress 무료. 광고 비즈니스에 결정적. S3 API 호환.
