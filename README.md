# Claude Ad Terminal

> **터미널이 일하는 시간, 가만 두지 마세요.**
> Claude 가 코드를 짜는 동안 짧은 광고 한 편이 흐르고, 작업이 끝날 즈음엔 잔고가 자라 있다.

---

## 1. 이게 뭔가

**Claude Code 터미널 래퍼 데스크탑 앱.** 시청자는 Claude Code 를 이 앱 안에서 실행한다.
터미널은 항상 풀스크린. **Claude 가 작업 중 (busy) 일 때만** 그 위에 광고가 덮인다. Claude 가 idle 로 돌아오면 광고는 사라지고 터미널이 다시 보인다. 위젯·브라우저 확장·MCP 플러그인 아님 — **터미널 그 자체를 우리 앱이 감싼다.**

```
[ 평소 — Claude idle ]                  [ Claude busy ]
┌──────────────────────────────┐        ┌──────────────────────────────┐
│  $ claude "리팩토링 해줘"     │        │ ╔══════════════════════════╗ │
│  > 분석 중...                 │   ━━▶  │ ║                          ║ │
│  > 적용 완료                  │  busy  │ ║   광고 영상              ║ │
│                              │        │ ║   (terminal 위에 오버레이)║ │
│  $ _                         │        │ ║                          ║ │
│                              │   ◀━━  │ ║   [자세히 보기 →]        ║ │
│                              │  idle  │ ╚══════════════════════════╝ │
│  잔액 ₩12,340                │ 복귀   │  잔액 ₩12,340                │
└──────────────────────────────┘        └──────────────────────────────┘
   풀스크린 터미널만 보임                광고가 터미널을 덮음
```

광고주는 별도로 웹 콘솔 (`/advertiser`) 에서 영상을 올리고 예산을 관리한다.

## 2. 한 줄 그림

```
┌─────────────────┐      광고(영상)       ┌────────────────────┐
│  광고주         │  ───────────────▶    │  시청자 (개발자)    │
│  (웹 콘솔)      │                       │  터미널 래퍼 앱     │
└────────┬────────┘                       └──────────┬─────────┘
         │                                            │
         │ CPM 결제                                   │ 토큰 적립
         │                                            │ + CTA 보상
         ▼                                            ▼
   "개발자에게 닿고 싶다"                      "기다리는 시간이 부수입"
```

세 사람이 다 이기는 구조 한 가지를 중심으로 설계됨:

- **시청자**: 어차피 Claude 가 작업하는 동안 기다리는 시간. 그 시간에 잔고가 늘어남.
- **광고주**: 개발자 대상 도달은 비싸고 어렵다. 여긴 청중이 100 % 개발자.
- **Claude 사용자**: 어차피 보는 광고면 보상받고 보자.

## 3. 누가 오는가

### 시청자 (Viewer) — Claude Code 를 매일 쓰는 개발자

| 페르소나 | 동기 |
|---|---|
| **사이드 프로젝트로 코딩하는 학생/주니어** | 적립이 곧 부수입. 학습 시간이 돈이 됨 |
| **하루 종일 Claude Code 를 켜둔 시니어** | 백그라운드로 알아서 돌아감. 작업 흐름 안 끊김 |
| **에이전트 워크플로우 돌리는 사람** | Claude 가 일하는 시간이 길수록 적립이 큼 |

> **공통점:** 광고 자체가 목적이 아님. 광고를 *어쩌다 보게 된다면* 그 시간에 작은 보상을 받고 싶을 뿐.

### 광고주 (Advertiser) — 개발자 청중을 사야 하는 곳

| 페르소나 | 동기 |
|---|---|
| **개발자 도구 SaaS** (DB, 모니터링, IDE, LLM) | 일반 광고망에선 노이즈. 여긴 청중이 전부 개발자 |
| **클라우드/인프라 서비스** | 결정권자가 직접 보는 경험 |
| **기술 채용** | 직군 정조준, 짧은 영상으로 임팩트 |
| **개발자 컨퍼런스/부트캠프** | 시기성·지역성 캠페인 |

> **공통점:** "개발자 청중" 이 비싸고 흩어져 있다는 걸 안다. 페이스북/구글 광고는 너무 광활. 여긴 좁고 정확.

## 4. 무엇이 있는가 (기능)

### 4-1. 시청자

```
설치 ──▶ 로그인 ──▶ 광고 시청 ──▶ 적립 ──▶ 출금
```

| 기능 | 설명 |
|---|---|
| **터미널 래퍼 앱** | Claude Code 가 안에서 돌아가는 Electron 데스크탑 앱. Mac · Windows · Linux. 평소 쓰던 그대로 작업 |
| **광고 오버레이** | 터미널이 항상 풀스크린. Claude 가 busy 일 때만 그 위에 광고가 덮이고, idle 로 돌아오면 사라짐 |
| **GitHub 로그인** | 별도 회원가입 절차 0. 개발자 계정 그대로 |
| **자동 광고 노출** | Claude 가 작업 중인 (busy) 시간에만 짧은 영상 한 편 — 어차피 코드 읽거나 기다리던 순간 |
| **시청 보상** | 끝까지 보면 토큰 적립. CTA 클릭 시 추가 적립 |
| **대시보드** | 잔액, 적립 내역, 일일 적립량 한눈에 |
| **출금** | 누적 토큰을 현금성 가치로 인출 |
| **레퍼럴** | 친구 초대 시 양쪽 보너스 적립 |
| **사생활 보호** | 시청 여부·시간만 기록. 키입력·작업 내용은 0 |

### 4-2. 광고주 (웹 콘솔, 데스크탑 앱 아님)

```
가입 ──▶ 영상 업로드 ──▶ 예산 충전 ──▶ 자동 노출 ──▶ 통계
```

| 기능 | 설명 |
|---|---|
| **GitHub 로그인** | 시청자와 같은 계정이어도, 광고주 등록은 별개 |
| **광고주 콘솔 (`/advertiser`)** | 잔액·광고 목록·통계 단일 페이지 |
| **영상 광고 등록** | mp4/webm 직접 업로드 (Cloudflare R2 호스팅) |
| **CTA 설정** | "지금 가입" 같은 행동 유도 + 외부 링크 |
| **CPM 단가** | 1,000 회 노출당 단가 자유 설정 |
| **예산 캡** | 총 예산 한도. 도달 시 자동 일시정지 |
| **활성 / 일시정지 토글** | 광고별 즉시 on/off |
| **노출 통계** | 광고별 누적 노출 수, 잔여 예산 |
| **예산 충전** | 광고주 콘솔에서 잔액 충전 |
| **노출 스케줄** | 시간대·일일 한도 등 세밀 조절 |

### 4-3. 운영

| 기능 | 설명 |
|---|---|
| **광고 검수** | 신규 광고는 admin 승인 후 노출 (부적절 콘텐츠 차단) |
| **출금 승인** | 시청자 출금 요청 검토 후 송금 (어뷰징 방지) |
| **유저/광고주 차단** | 정책 위반 계정 정지 |

## 5. 핵심 약속

| 약속 | 의미 |
|---|---|
| **작업을 끊지 않는다** | Claude 가 busy 일 때만 덮이고, 입력 받을 차례 (idle) 가 되면 사라짐. 키 입력을 가로막지 않음 |
| **개발자 청중만 있다** | 일반 트래픽 광고망과 다름. 모든 시청자가 Claude Code 사용자 |
| **사생활은 최소 정보만** | 시청/CTA 여부 외에는 수집 안 함 |
| **CPM 결제, 결과는 투명하게** | 광고주에게 노출 수가 그대로 보임. 블랙박스 없음 |

## 6. 자주 묻는 질문

**Q. 기존 터미널/IDE 와 무슨 관계인가요?**
시청자용 앱은 **Claude Code 터미널 래퍼** 입니다. iTerm / Terminal.app / VSCode 내장 터미널을 대체하는 별도 데스크탑 앱이에요. 위젯 · 브라우저 확장 · MCP 플러그인 아닙니다. 앱 안에 터미널 영역 + 광고 영역이 한 창으로 붙어 있습니다.

**Q. 광고는 언제 떠요?**
Claude 가 작업 중 (busy) 일 때만 풀스크린 터미널 위에 덮입니다. Claude 가 입력 받을 차례 (idle) 가 되면 사라지고 터미널이 다시 보입니다. 어차피 코드 읽거나 기다리던 순간만 사용됨.

**Q. 키 입력이나 작업 내용을 수집하나요?**
아니요. 광고 노출에 필요한 최소 정보 (시청 여부·시간·CTA 클릭) 만 기록합니다.

**Q. 토큰은 어디서 쓰나요?**
대시보드에서 출금 신청. 누적 토큰을 현금성 가치로 인출합니다.

**Q. 광고주는 어떻게 시작하나요?**
`/advertiser` 페이지에서 GitHub 로 가입 → 영상 한 편 업로드 → 잔액 충전 → 자동 노출.

**Q. CTA 를 클릭하면 보상이 더 큰가요?**
네. 시청만으로도 기본 보상이 적립되고, 광고의 CTA 액션을 수행하면 추가 토큰이 함께 적립됩니다.
